# 사용 가이드

!!! tip "딱 세 가지만 기억하세요"
    1. 시작할 때 **기본 씬(`02 mua-default-scenes`) 불러오기**
    2. 끝나면 **다시 `02 mua-default-scenes` 불러오기**
    3. 그 사이에 믹서의 게인·연결을 **함부로 만지지 마세요.** (본인 기타 채널만 예외 — 3번 참고)

## 1. 시작 전 핵심

- 세팅은 이미 다 잡혀 있습니다. 여러분이 할 일은 **씬을 불러오고, 본인 악기를 기준에 맞추는 것**뿐입니다.
- 채널 연결, 음량 밸런스, 모니터 분배는 씬 안에 저장돼 있어 직접 손댈 필요가 없습니다.

## 2. 씬 불러오기

평소 합주는 항상 **`02 mua-default-scenes`** 씬을 씁니다.

1. 믹서(검은 콘솔)에서 **SCENES**(씬 목록) 버튼을 누릅니다.
2. 목록에서 **`02 mua-default-scenes`**를 선택합니다.
3. **LOAD**(불러오기)를 누릅니다.
4. 합주가 끝나면 다시 **`02 mua-default-scenes`**를 불러와 원상복구합니다.

!!! warning "이 씬들은 불러오지 마세요"
    - **`00`** (최초 기본세팅, 추정): 합주실 최초 세팅으로 추정되지만 검증되지 않았습니다. 평소 쓰지 마세요.
    - **`01 backup_before_260530`**: 이 가이드가 나오기 전(2026-05-30 기준) 세팅입니다. **꼬인 부분이 많아** 복구용으로만 보관 중이니, 불러오지 마세요.

## 3. 악기별 가이드

악기마다 기준 위치가 정해져 있습니다. 아래 기준에서 벗어나지 않게 맞춰 주세요.

### 베이스 (Fender Rumble)

- 앰프 마스터 볼륨을 **12시에 고정**합니다.
- 앰프 캐비넷(앰프에서 나오는 소리)과 PA가 함께 나갑니다. 마스터를 올리면 PA로 가는 소리도 같이 커져 전체 밸런스가 깨집니다.
- 12시에서 움직이지 말고, 본인 모니터 음량은 **4번(스피커 조절)** 방법으로 맞추세요.

### 키보드

- 본체 볼륨 노브를 **12시 기준**으로 둡니다. (이 기준으로 세팅돼 있습니다.)

### 기타 (Marshall JVM)

기타는 연결 방식에 따라 X32 채널 게인을 맞춰야 합니다. **이 채널 게인 조정만 예외적으로 이용자가 직접 할 수 있습니다.** (그 외 게인·노브는 건드리지 마세요.)

| 연결 방식 | X32 채널 게인 | 추가 안내 |
|----------|-------------|----------|
| 앰프 인풋에 꽂기 (기본) | 0 dB (씬 기본값) | 앰프 채널 볼륨: 클린·크런치 12시 / OD 10시 |
| 멀티이펙터를 XLR로 직접 PA에 (앰프 안 씀) | -10 dB | 멀티 출력 볼륨으로 미세조정 |
| 멀티이펙터를 앰프 Return에 직결 | +10 dB | 멀티 출력 볼륨으로 미세조정 |

- 단순 페달만 쓴다면 **앰프 인풋(기본)** 을 사용하세요.

!!! warning "튜브앰프 주의"
    Marshall JVM은 튜브앰프입니다. 스탠바이를 끄고 PA만 쓸 때도 **스피커(캐비넷) 선을 빼지 마세요.** 부하 없이 켜면 앰프가 손상됩니다. 또한 단순 페달은 앰프 Return(Power Amp Insert)에 직접 꽂지 말고 **앰프 인풋**에 연결하세요.

### 보컬

- 마이크를 본인 자리 채널에 꽂으면 됩니다. (게인은 이미 세팅돼 있습니다.)
- **안 쓰는 마이크는 음소거(mute)** 하세요. 하울링(삐 소리)을 막아 줍니다.

## 4. 스피커 조절 (음량 조정 방법)

### 메인 스피커 (방 전체로 나가는 소리)

- 콘솔의 **메인 노브(Main 마스터)** 로 전체 음량을 조절합니다.
- 악기별 밸런스는 **각 채널 페이더(볼륨 막대)** 로 미세조정합니다.

### 모니터 스피커 (본인 자리 앞 바닥)

그냥 페이더를 올리면 메인 소리가 바뀝니다. 본인 모니터만 조절하려면 두 단계가 필요합니다.

1. 콘솔에서 **본인 모니터 버스를 선택**합니다. (Sends on Fader / 버스 셀렉트)
2. 그 상태에서 **왼쪽 채널 페이더로 각 악기를 조절**합니다.

이 두 단계를 거쳐야 본인 모니터 안에서 악기별 세부 조정이 됩니다.

## 5. 문제가 생기면

순서대로 확인하세요.

1. 악기·앰프 전원과 **케이블**이 제대로 꽂혔는지
2. 해당 채널이 **음소거(mute)** 되어 있지 않은지
3. 채널 **페이더(볼륨 막대)** 가 내려가 있지 않은지
4. 그래도 안 되면 **`02 mua-default-scenes`를 다시 불러오기**

그래도 해결되지 않으면 관리자에게 연락하세요. (아래 7번 문의 참고)

## 6. 하지 말 것

!!! danger "건드리지 마세요"
    - 입력/출력 **연결(라우팅·패치)** 변경
    - **48V(팬텀파워)** 켜고 끄기
    - 씬 **저장(SAVE/덮어쓰기)** — 불러오기(LOAD)만 하세요
    - **다른 사람 채널의 게인**이나 컴프·게이트·EQ 건드리기 (본인 기타 채널 게인을 3번 표대로 맞추는 것만 예외)
    - 베이스 앰프 마스터를 **12시 이상**으로 올리기
    - 모니터/메인 **버스 라우팅** 변경

    바꾸면 다음 사람의 세팅이 꼬입니다. 무언가 바꿨다면 **반드시 `02 mua-default-scenes`를 다시 불러오세요.**
