# 사용 가이드

!!! tip "딱 세 가지만 기억하세요"
    1. 시작할 때 **기본 씬(`02 mua-default-scenes`) 불러오기**
    2. 끝나면 **다시 `02 mua-default-scenes` 불러오기**
    3. 그 사이에 믹서의 게인·연결을 **함부로 만지지 마세요.** (기타 채널만 예외 — 3번 참고)

> 이 합주실은 공용이라 세팅이 자주 흐트러집니다(스피커 전원 꺼짐, 볼륨 변경, 믹서 뒷편 연결 바뀜 등). 그래서 **2026.5.31 기준 공용 세팅(`02 mua-default-scenes` 씬)**을 정해 두었습니다. **씬을 지키는 것**이 가장 중요합니다. 자세한 배경과 정상화 내역은 [홈](index.md)을 참고하세요.

## 이 합주실 구성

- 드럼을 제외한 모든 악기(베이스·키보드·기타·보컬)는 **믹서를 거쳐 방 전체로 나가는 메인 스피커**로 소리가 나옵니다.
- 동시에 각 연주자는 **본인 자리 앞 바닥의 모니터 스피커**로, 자기 악기가 좀 더 강조된 소리를 따로 들을 수 있습니다.
- 드럼은 마이크를 쓰지 않고 **생음 그대로** 들립니다.
- 그래서 합주는 **본인 악기를 믹서에 연결하고, 본인 모니터에서 자기 소리를 키워 들으며** 함께 맞추는 구조입니다.

!!! info "드럼 마이크는 왜 없나요?"
    - 합주실 초기 세팅에는 있었던 것으로 보이지만, 마이크 일부 분실과 연결 꼬임으로 현재는 정상 사용이 어렵습니다.
    - 다행히 현재 합주실 규모에서는 크게 필요하지 않습니다.


## 1. 시작 전 핵심

- 세팅은 이미 다 잡혀 있습니다. 여러분이 할 일은 **씬을 불러오고, 본인 악기를 기준에 맞추는 것**뿐입니다.
- 채널 연결, 음량 밸런스, 모니터 분배는 씬 안에 저장돼 있어 직접 손댈 필요가 없습니다.

## 2. 씬 불러오기

평소 합주는 항상 **`02 mua-default-scenes`** 씬을 씁니다.

1. 믹서(검은 콘솔)에서 **SCENES**(씬 목록) 버튼을 누릅니다.
2. 목록에서 **`02 mua-default-scenes`**를 선택합니다.
3. **LOAD**(불러오기)를 누릅니다.
4. 합주가 끝나면 다시 **`02 mua-default-scenes`**를 불러와 원상복구합니다.

!!! warning "참고용 구 씬들"
    - **`00`** (최초 기본세팅, 추정): 합주실 최초 세팅으로 추정되지만 검증되지 않았습니다.
    - **`01 backup_before_260530`**: 이 가이드가 나오기 전(2026-05-30 기준) 세팅입니다. **꼬인 부분이 많아** 복구용으로만 보관 중입니다.

## 3. 악기별 가이드

악기마다 기준 위치가 정해져 있습니다. 아래 기준에서 벗어나지 않게 맞춰 주세요.

### 베이스 (Fender Rumble 500)

- 앰프의 마스터 볼륨을 **12시 근처에서 조정**합니다.
- 베이스는 앰프 볼륨(캐비넷에서 나오는 소리)도 함께 나가므로, **12시를 기준점**으로 두되 그 근처에서 무대음을 맞춰 조정하세요.
- 단, 마스터를 크게 올리면 **PA(메인 스피커)로 가는 소리도 같이 커져** 전체 밸런스가 깨집니다. 12시에서 크게 벗어나지 마세요.

### 키보드

- 키보드 2대 모두 본체 볼륨 노브를 **12시 근처에서 조정**합니다.

### 기타 (Marshall JVM 410H/C)

기타는 연결 방식에 따라 **믹서의 기타 인풋 채널 게인**을 맞춰야 합니다. **이 채널 게인 조정만 예외적으로 이용자가 직접 할 수 있습니다.** (그 외 게인·노브는 건드리지 마세요.)

**게인 조절 방법**

1. 믹서 왼쪽 채널 묶음에서 **`건반/기타`** 버튼을 누릅니다. 그 안에서 **5번째 = 기타1, 6번째 = 기타2** 입니다. (믹서에 붙은 스티커 라벨 기준)
2. 채널 윗쪽의 **gain 노브를 돌려** 조절합니다.
3. 우측 콘솔의 **HOME(메인 화면)을 누르면 정확한 gain dB 숫자**가 표시됩니다. 그 숫자를 보면서 아래 표 값에 맞추세요.

| 연결 방식                                | 믹서 기타 채널 게인 | 추가 안내                                                                        |
|--------------------------------------|-------------|------------------------------------------------------------------------------|
| 앰프 인풋에 꽂기 (기본)                       | 0 dB (씬 기본값) | 앰프 채널 볼륨: 클린·크런치 12시 / OD 10시 전후에서 조정                                       |
| 멀티이펙터를 XLR로 직접 PA에 (앰프 안 씀)          | -10 dB | 앰프 뒷면의 **line out 케이블을 빼서** 연결합니다. 멀티이펙터 마스터 볼륨으로 조정                         |
| 멀티이펙터를 앰프 Return에 직결                 | +10 dB | 멀티이펙터 마스터 볼륨으로 조정                                                            |
| 4CM 구성 (기타 -> 페달 -> 앰프 -> 페달 -> 캐비넷) | -10 ~ 0 dB (미터 보며 조절) | 페달 구성마다 레벨이 달라 고정값이 없습니다. 인풋 채널의 시그널 미터를 보며 맞추세요. (강한 리드 톤이 -6~-12dB 정도 들어오는 수준) |

- 단순 페달만 쓴다면 **앰프 인풋(기본)** 을 사용하세요.

**캐비넷(앰프에서 직접 나오는 소리)은 켜도 되고 안 켜도 됩니다. 안 켜도 충분합니다.** 취향대로 자유롭게 쓰세요. 켜려면 앰프 **STANDBY를 ON**으로 하면 됩니다. 캐비넷 음량은 **앰프 마스터 볼륨**으로 조정합니다. (PA로 나가는 소리와는 별개로, 무대에서 본인 귀로 직접 듣는 앰프 소리입니다.)

!!! warning "멀티이펙터를 XLR로 직접 PA에 연결했다면"
    앰프 뒷면 **line out 케이블을 다시 연결**해 **반드시 원상복귀**하세요. 빼 둔 채로 두면 다음 사람이 앰프를 못 씁니다.

### 보컬

- 현재 마이크는 **1개만 비치**돼 있고, **MIC2** 채널에 연결돼 있습니다. (게인은 이미 세팅돼 있어 그대로 쓰면 됩니다.)
- MIC1·MIC3용 XLR 잭도 있지만 지금은 바닥에 정리되지 않은 상태입니다. (마이크를 추가로 비치하기 전까지는 미사용)
- **안 쓰는 마이크 채널은 음소거(mute)** 하세요. 하울링(삐 소리)을 막아 줍니다.

**보컬 리버브(울림) 조절**

1. 믹서 왼쪽에서 **`리버브`** 라벨(Bus 9~16)을 누릅니다.
2. 그 안의 **5번째 페이더**(= Bus13, 현재 MIC2 보컬 리버브)를 **-15 근처**에서 취향껏 올리고 내립니다.

## 4. 스피커 조절 (음량 조정 방법)

!!! info "스피커 전원"
    메인·모니터 스피커 전원은 합주실 **일괄 전원 스위치**로 한 번에 켜지고 꺼집니다. **개별 스피커의 전원 버튼은 건드리지 마세요.** (개별로 꺼 두면 다음 사람이 "소리가 안 난다"고 헤매게 됩니다.)

!!! warning "스피커 본체 설정은 건드리지 마세요 (메인·모니터 공통)"
    음량은 **믹서에서만** 조절합니다 (메인은 메인 노브, 모니터는 버스 페이더 — 아래 참고). 스피커 **본체의 볼륨·노브는 고정값**입니다.

    - **메인 스피커**: CH01 Vol = FULL / MAIN Vol = -10dB 고정
    - **모니터 스피커**: 본체 볼륨·설정 그대로 두기

### 메인 스피커 (방 전체로 나가는 소리)

- 콘솔의 **메인 노브(Main 마스터)** 로 전체 음량을 조절합니다. (-10dB ~ 0 권장)
- 악기별 밸런스는 **각 채널 페이더(볼륨 막대)** 로 조정합니다. (-15dB 근처 권장)

### 모니터 스피커 (본인 자리 앞 바닥)

**모니터 전체 음량**

- 본인 **모니터 스피커 버스 페이더**를 올리고 내려 조절합니다.

!!! note "현장에서 채울 것"
    권장 음량 값은 추후 기재 예정입니다.

**각 악기별로 따로 조절하고 싶을 때** (현재 값: 본인 악기 -10 dB / 다른 악기 -18 dB)

!!! warning "굳이 조절하지 않아도 됩니다 (숙련자에게만 권장)"
    - 이미 본인 악기가 8dB 정도 강조돼 있습니다.
    - 메인 스피커로 가는 각 악기 페이더 조정과는 별개입니다.

1. 본인 모니터링 스피커(버스)를 선택합니다.
2. **send on fader** 버튼을 누릅니다. (버튼이 빨갛게 깜빡입니다.)
3. 각 악기 페이더를 올리고 내려 조절합니다.
4. 다 됐으면 **send on fader 버튼을 다시 눌러 해제**합니다.

## 5. 문제가 생기면

순서대로 확인하세요.

1. 악기·앰프 전원과 **케이블**이 제대로 꽂혔는지, 켜졌는지
2. 해당 채널이 **음소거(mute)** 되어 있지 않은지
3. 채널 **페이더(볼륨 막대)** 가 내려가 있지 않은지
4. 그래도 안 되면 **`02 mua-default-scenes`를 다시 불러오기**

## 6. 하지 말 것

!!! danger "건드리지 마세요"
    - 입력/출력 **연결(라우팅·패치)** 변경
    - **48V(팬텀파워)** 켜고 끄기
    - 씬 **저장(SAVE/덮어쓰기)** — 불러오기(LOAD)만 하세요
    - **다른 사람 채널의 게인**이나 컴프·게이트·EQ 건드리기 (본인 기타 채널 게인을 3번 표대로 맞추는 것만 예외)
    - 모니터/메인 **버스 라우팅** 변경
    - 메인 스피커, 모니터링 스피커 자체 세팅 변경
    - 스피커(메인·모니터) **개별 전원** 켜고 끄기 — 전원은 합주실 **일괄 전원 스위치**로만 관리합니다.

    바꾸면 다음 사람의 세팅이 꼬입니다. 무언가 바꿨다면 **반드시 `02 mua-default-scenes`를 다시 불러오세요.**

## 7. 아직 확인 중인 것들

다음은 아직 정리·확인 중이라 가이드에 넣지 않았습니다. 사용이 필요하면 관리자에게 문의해 주세요.

- 어쿠스틱 기타 입력 (DI)
- 블루투스 입력
- PC 입력
- 부밍(웅~ 하는 울림) 측정 및 EQ 조절
