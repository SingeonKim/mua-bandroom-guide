## 1. 시스템 구성 요약

|구분     |장비                    |수량        |
|-------|----------------------|----------|
|콘솔     |Behringer X32 Producer|1         |
|모니터 스피커|Yamaha DHR12M         |5 (세션별 바닥)|
|메인 스피커 |JBL PRX ONE (L/R)     |2         |

**신호 흐름 큰 그림**

```
입력 채널 ──┬─→ Bus1~5 (모니터, PRE-fader) → 각 세션 바닥 모니터
            ├─→ Main L/R (POST-fader)        → 메인 스피커
            └─→ Bus13 (FX 리버브, 보컬만)     → 리버브 → return
```

- **모니터 = Pre-fader** (메인 페이더 만져도 모니터 안 변함)
- **메인 = Post-fader** (채널 페이더로 메인 밸런스 조절)
- 둘이 독립 → 한쪽 만져도 다른 쪽 안 흔들림

## 1-1. 물리 입출력 패치 (믹서 뒷면)

> 출처: 믹서(X32 Producer) 뒷면에 부착돼 있던 **초기 세팅 라벨 문서** + 작성자가 정리한 소프트 채널 매핑.
> **물리 잭 번호와 채널 번호가 다른 이유**: X32 **Input Routing(내부 입력 패치)** 에서 잭을 원하는 채널에 할당했기 때문이다. 즉 뒷면 물리 잭 ↔ 콘솔 채널은 1:1 고정이 아니라, 라우팅으로 연결돼 있다. **채널 번호는 X32-Edit의 `Routing > Input`에서 확인/변경**한다. (물리 결선과 채널 매핑은 둘 다 맞으며 모순이 아니다.)

!!! note "ch17·ch19 보컬 마이크 미비치"
    **ch17·ch19는 보컬 마이크용으로 채널이 배정돼 있으나, 현재 실물 마이크 장비가 없어 사용할 수 없다.** 채널 세팅값은 마이크 비치를 대비해 기록·보존한다. 실물 마이크를 비치하면 그대로 사용 가능. (ch18은 현재 사용 가능)
- ※ 키보드: 48V 불필요(라인레벨). **본체 볼륨 노브 12시 기준**(이 기준으로 믹서 게인 잡음). **Fantom(ch9/10)은 거리가 멀어 DI박스 경유** → DI가 마이크레벨로 감쇠시키므로 **믹서 게인을 높게** 잡아야 정상 레벨. 48V는 음량과 무관(패시브 DI면 안 먹는 게 정상). **게이트 OFF**(라인 입력이라 잡음 안 새고, 게이트가 서스테인/패드 여운을 자름).
- ※ 기타·베이스 게이트(약): 클린에서도 노이즈가 조금 있어 메인/모니터에 거슬릴 때 **아주 약하게**. Threshold -55dB(안 닫히면 -50까지), **Range -10~-15dB(완전차단 X)**, Attack 0.5~2ms, Hold 베이스 30~50·기타 20~40ms, Release 베이스 200~300·기타 150~250ms. “노이즈 겨우 닫히는 최저 threshold”가 베스트(올릴수록 약한 연주음 잘림).
- ※ **노이즈 근본 점검 (게이트 전에)**: 그라운드 험이면 ① 그라운드 리프트(Rumble/DI) ② 전원 같은 멀티탭/콘센트로 모으기 ③ 의심 케이블 교체. 게이트는 증상 완화일 뿐 — 연주 중엔 노이즈 그대로 들리므로 원인 제거 우선.
- ※ **보컬 마이크 입력 경로 (정정)**: 보컬 마이크는 **뒷면 로컬 입력 XF13~16(MIC#1~4)** 에 연결되고, X32 **Input Routing으로 ch17~19에 매핑**된다. (이전의 "하단 판넬 AES50 스테이지박스 추정"은 **철회** — 뒷면에 AES50 A/B·ULTRANET 포트는 존재하나 현재 외부 스테이지박스 결선은 없음.) → 1-1 물리 패치 참고.
- ※ **어쿠스틱기타 DI / 블루투스 수신기 / 노트북 DI (갱신)**: 모두 물리 입력에 이미 배정돼 있음(**XF11 어쿠스틱DI, XF12 블루투스, TRS5/6 노트북DI**). 현재 미사용. 사용 시 채널 라우팅 후 입력 행 추가. → 1-1 물리 패치 참고.

## 2-1. 기타/베이스 앰프 — 연결 케이스 & 게인 기준

> 기타(JVM 410H/410C), 베이스(Rumble 500)는 연주자가 **어떻게 연결하느냐에 따라 PA로 가는 신호 레벨이 달라짐**.
> 같은 채널(ch13/14/8)에 들어와도 케이스마다 적정 게인이 다르므로, **케이스별 게인값을 고정해두고 사용자는 본인 케이스값으로 맞추기만** 한다 (게인 외 미세조정은 장비/앰프 쪽에서).
> 게인값은 **현장 실측 기준** (페이더 -15 기준, 기타1 JVM410H 측정).

### 핵심 원칙 (왜 이렇게 나누나)

- **소리 경로 2개**: ① 캐비넷(앰프 스피커=무대음) ② PA(라인아웃→믹서→메인/모니터)
- “캐비넷만” 쓰면 PA를 안 거침 → **믹서/모니터 세팅과 무관** (이 문서 범위 밖)
- 합주 시엔 **PA로 보내기 권장** (안 그러면 다른 세션 모니터에 그 파트가 안 들어감)

### 연결 4가지 케이스 (PA 게인 실측 기준)

|-------------------------|----------------------------------------------|:------:|--------|:-----------------:|-------------------------------------------------------------------|
|**A. 앰프 인풋**             |악기→앰프 인풋→캐비넷+라인아웃                             |O       |앰프 라인아웃 |**0 dB**           |앰프 채널 인풋 볼륨 (클린·크런치 12시 전후 / OD 10시 전후)                            |
|**B. 4CM (개별 페달)**       |악기→앞단 페달→앰프 인풋 / 앰프 SEND→뒷단 페달→RETURN→캐비넷+라인아웃|O       |앰프 라인아웃 |**-10 ~ 0 dB (조절)**|페달 구성마다 레벨이 달라 고정 불가. 미터 보며 게인 조절.|
|**C. 멀티→Return 직결**      |악기→멀티(프리/캡시뮬)→앰프 1/4” RETURN→파워앰프+캐비넷+라인아웃    |O       |앰프 라인아웃 |**+10 dB**         |멀티이펙터 출력 볼륨                                                        |
|**D. 멀티 XLR 직결 (No Amp)**|악기→멀티(캡시뮬 ON)→XLR→PA                          |X       |멀티 라인아웃 |**-10 dB**         |멀티이펙터 출력 볼륨                                                        |

* JVM은 라인아웃이 이펙트 루프보다 뒤(마스터 직전)에 있어 **Return 직결 신호가 라인아웃에 잡힘** (매뉴얼+실측 확인). **베이스(Rumble)는 Return 직결을 쓰지 않음 → C 케이스는 기타에만 해당.**

> **PA 게인은 케이스별 3값으로 고정**:
>
> - **D 멀티 XLR 직결 → -10dB** (XLR 라인레벨이라 hot, 게인 낮춤)
> - **A 앰프 인풋 → 0dB** (앰프 프리 풀로 거친 라인아웃)
> - **B 4CM → -10~0dB 조절** (페달 구성마다 레벨이 달라 고정 불가, 사용자가 미터 보며 조절 — 4케이스 중 유일하게 게인 조절 필요)
> - **C Return 직결 → +10dB** (멀티 출력이 앰프 뒷단만 거쳐 더 작게 나옴)
> - D와 C가 **약 18dB 차이** (라인레벨 +4dBu vs 인스트레벨 -10dBV + 경로차) — 실측으로 확인됨
> - 캐비넷(무대음)이 필요하면 **마스터 볼륨으로 조절** (PA와 독립, A·B·C 모두 가능)

!!! warning "페달만 쓰는 사용자 주의"
    - **Return 직결(C)은 출력 레벨 조절이 되는 멀티이펙터/모델러 전용.** 페달은 출력 볼륨이 없어 +10dB 게인을 맞출 수 없음.
    - **단순 페달 사용자는 앰프 인풋(A)을 사용** → 앰프 채널 볼륨으로 조절. 페달을 Return에 직접 꽂지 말 것.

### 케이스별 주의사항

**JVM 410H / 410C (기타1·2)**

*캐비넷 / PA 운영 가이드 (연주자 안내용)*

- **캐비넷 음량 = 마스터 볼륨으로 조절.** 라인아웃(PA)과 독립이라 무대음을 자유롭게 키우고 줄여도 PA 밸런스 안 흔들림.
- **캐비넷 안 쓸 거면 = 스탠바이 OFF → PA(모니터+메인)만 활용.**

!!! danger "튜브앰프 — 스피커선 빼지 말 것"
    **스탠바이 끄고 PA만 쓸 때도 캐비넷/스피커는 연결 유지** (튜브앰프는 부하 없이 켜면 출력 트랜스포머 손상. 410C 콤보는 내장 스피커가 부하 → 스피커선 빼지 말 것).

*라인아웃(PA) 레벨 조절 — 어디서 되고 안 되나*

|무엇으로          |캐비넷|라인아웃(PA)          |
|--------------|:-:|:----------------:|
|마스터 볼륨        |O  |**X (영향 없음)**     |
|채널 Gain/Volume|O  |O (둘 다 같이 변동)     |
|믹서 게인/PAD     |X  |**O (PA만 여기서 조절)**|

- 라인아웃은 마스터로 못 줄임. 채널 게인/볼륨으로 줄이면 캐비넷도 같이 줄어듦.
- → **PA 레벨은 앰프 말고 믹서(X32 게인/PAD)에서 조절**하는 게 정석.

*기술 주의*

- 라인아웃이 **hot** → PA 게인은 케이스값(직결 -10 / 앰프인풋·4CM 0 / Return +10)에서 시작, 미터로 미세조정.
- **루프 2개 구분**:
  - 4CM 뒷단 페달(B) → **①Serial/Parallel FX Loop**. 페달이면 딥스위치 **-10dBV** / 라인레벨 멀티면 +4dBu.
  - Return 직결(C) → **②Power Amp Insert의 1/4” RETURN**. 라인레벨 고정, 딥스위치 무관.

!!! danger "페달을 Power Amp Insert에 꽂지 말 것"
    **페달류를 ②Power Amp Insert에 꽂으면 신호 과다로 크래클 노이즈** → 페달은 ①, 라인레벨 장비는 ②

**Fender Rumble 500 (베이스)**

*권장: 캐비넷 + PA 동시 (Case A)*

- 베이스는 **캐비넷 푸시(저역 공기 진동)가 톤의 핵심** → 무대음 같이 켜는 게 정석. Rumble 500은 무대음용으로 충분히 좋은 앰프.
- **기본 세팅값 (실측): PA 채널 게인 +21dB / 페이더 -15 / 앰프 마스터 12시**
- 라인아웃(XLR) → PA + 내장 스피커 무대음 동시. 마스터=PA 연동이라 “무대 톤=PA 톤” 일치가 장점.

*특수: 캐비넷 끄기 (Case B) — 필요시만*

- **헤드폰 잭(1/8”)에 더미 플러그** 꽂으면 내장 스피커 뮤트, 라인아웃은 유지 (실물 동작 확인). 늦은 시간 등 무대음 빼야 할 때만.
- 솔리드스테이트(Class-D)라 스피커 꺼도 부하 걱정 없음. **스탠바이 없음(정상).**

*기술 주의*

- 라인아웃이 **post everything** → **마스터 볼륨 올리면 PA로 가는 신호도 같이 커짐**.
  - → “내 소리 작아”로 마스터 올리면 PA 베이스도 커져 전체 밸런스 깨짐.

!!! danger "베이스 마스터 12시 고정"
    **마스터 12시 고정.** 본인 모니터는 Bus send로 조절. (라인아웃이 post everything이라 마스터를 올리면 PA 베이스도 같이 커져 전체 밸런스가 깨짐.)

- 라인아웃/이펙트 루프에 **레벨 조절 없음** → 앰프 톤·EQ가 그대로 PA로. 부밍하게 잡으면 PA도 부밍.
- 험 생기면 **그라운드 리프트 ON**. 고역 트위터는 후면 호른 on/off로 별도 조절 가능.

## 2-2. 보컬 마이크 세팅 (Shure PGA57 × 3, ch17/18/19)

> PGA57은 **다이나믹 카디오이드 악기용 마이크**. 보컬에 쓸 수는 있으나 전용 보컬용(SM58/PGA58)보다 저역·풍성함이 덜하고 팝필터가 약함 → **로우컷 + EQ 보정**으로 다듬어 사용.
> 다이나믹이라 **48V OFF, 출력 낮아 게인 넉넉히**. 카디오이드라 하울링엔 유리(메인 보컬 비중 높이는 셋업에 적합).
>
> ※ **현재 ch17·ch19 보컬 마이크는 미비치 상태(실물 마이크 없음)** — 아래 세팅은 마이크 비치 시 적용. ch18은 현재 사용 가능.

### 게인 (제일 먼저, 컴프 OFF 상태에서)

- 컴프 바이패스 → **제일 세게 부를 때 피크 -6~-10** (돌발 대비 -8~-10 여유). 평균 -18~-12.
- 보컬은 다이내믹 편차 큼 → “지를 때” 기준. PGA57 출력 낮아 게인 높이 올라가는 게 정상.

### 로우컷 (HPF)

- **100~120Hz ON** 확실히. 팝(ㅍ/ㅂ) 노이즈·저역 부밍·하울링 대응.

### EQ (악기용 → 보컬 보정, 출발점·솔로로 미세조정)

|대역       |조치    |이유               |
|---------|------|-----------------|
|150~250Hz|+2~3dB|보컬 두께 보충 (과하면 부밍)|
|400~600Hz|-2dB  |박스/먹먹함 제거        |
|3~5kHz   |+2~3dB|가사 명료도(프레즌스)     |
|8~10kHz  |귀로 판단 |거칠면 컷, 답답하면 부스트  |

### 컴프 (약하게)

|파라미터       |값                              |
|-----------|-------------------------------|
|Threshold  |GR **3~4dB** 물리게 (대략 -18~-24dB)|
|Ratio      |2:1 ~ 3:1                      |
|Attack     |10~20 ms                       |
|Hold       |0~10 ms                        |
|Release    |100~150 ms                     |
|Makeup Gain|+3~4dB (GR만큼 보충)               |
|Knee       |Soft                           |

### 게이트 (선택 — 합주실 생드럼/앰프 새는 게 거슬릴 때만)

|파라미터     |값                        |
|---------|-------------------------|
|Threshold|-40~-45dB (안 부를 때만 닫히게)  |
|Range    |-10~-20dB (완전차단 X, 자연스럽게)|
|Attack   |0.5~2 ms                 |
|Hold     |20~50 ms                 |
|Release  |100~250 ms               |

- 모르겠으면 **일단 OFF**. 잘못 잡으면 첫 글자 잘림.

### 적용 순서 & 복사

1. 게인(컴프 OFF, 피크 -6~-10) → 2. 컴프 ON, GR 3~4dB → 3. Attack/Release 조정(펌핑 확인) → 4. Makeup gain → 5. 게이트 필요시

- 3채널 다 PGA57 → 한 채널 잡고 **Copy/Paste**. 게이트 threshold만 자리(드럼 근접도)별 미세조정.

!!! warning "안 쓰는 마이크는 닫기"
    - 열린 마이크 수 ↑ = 하울링·잡음 ↑. **사용 안 하는 보컬 채널은 뮤트/페이더 다운.**
    - (확인 필요: 보컬 동시 사용 인원 — 상시 몇 명 부르는지)

## 3. 모니터 분배 (Bus1~5, PRE-fader)

> Yamaha DHR12M x5, 각 세션 앞 바닥
> 방식: **본인 소리 크게 + 나머지 파트 적당히** (합주 박자용)
> 작업법: `Sends on Fader` → Bus 선택 → 각 채널 send 양 조절

!!! warning "현재 운영 기준값 — 합주로 추가 미세조정될 수 있음"
    **현재 이 값으로 운영 중이며, 합주로 추가 미세조정될 수 있음.**

    - 각 Bus에서 **본인 악기 send = -10** / **다른 악기 send = -15** (Sends on Fader 기준)
    - 표의 “본인 -10 / 타 -15”는 현재 운영 기준값. 연주자 요청 따라 미세조정.

> **Bus1~5 용도 (확정)**: Bus1=기타1 / Bus2=베이스 / Bus3=기타2 / Bus4=건반 / Bus5=드럼. 각 Bus = 해당 악기 연주자 앞 모니터에 그 악기를 강조해 보내는 버스다.

!!! note "현장에서 채울 것"
    - **모니터 출력(Bus 마스터) 페이더 권장값 = `______`** (추후 기재, 방·세션별 음량 맞춤)

    |입력 \ Bus   |Bus1 기타1|Bus2 베이스|Bus3 기타2|Bus4 건반|Bus5 드럼|
    |-----------|:------:|:------:|:------:|:-----:|:-----:|
    |베이스(8)     |-15     |**-10** |-15     |-15    |-15    |
    |건반1·2(9-12)|-15     |-15     |-15     |**-10**|-15    |
    |기타1(13)    |**-10** |-15     |-15     |-15    |-15    |
    |기타2(14)    |-15     |-15     |**-10** |-15    |-15    |
    |보컬(17-19)  |-15     |-15     |-15     |-15    |-15    |

- **Bus1~5 용도는 위와 같이 확정** (Bus1=기타1 / Bus2=베이스 / Bus3=기타2 / Bus4=건반 / Bus5=드럼).
- 드럼은 생음 → 어느 모니터에도 “드럼 채널” 안 들어감
- Bus5(드럼 모니터)엔 드럼 본인 소리 없이 **다른 파트만** (드러머가 합 맞추게)
- 보컬은 모든 모니터에 -15로 깔아 다른 연주자가 박자/가사 맞추게. 보컬 본인 모니터는 메인 스피커로 대체(4번 참고).

### 모니터 스피커 본체 설정 (DHR12M)

- Input1 toggle: **LINE**
- Level: **0dB (12시)**
- D-CONTOUR: **MONITOR**
- HPF: **100/120Hz** (베이스 앞 모니터만 OFF or 100Hz)

### Bus → 물리 아웃풋 매핑 (Routing/Out)

> **물리 결선(확정, 초기 세팅 라벨)**: **XM1~5 = 모니터 5대(DHR12M), XM7 = Main_L, XM8 = Main_R, XM L/R = 컨트롤룸 모니터, XM6 = 미사용.** (→ 1-1 참고)
> **Bus↔XM 매핑(확정)**: 순서대로 1:1 대응한다.

| 물리 출력 | 받는 Bus | 모니터 용도 |
|----------|---------|------------|
| XM1 | Bus1 | 기타1 모니터 |
| XM2 | Bus2 | 베이스 모니터 |
| XM3 | Bus3 | 기타2 모니터 |
| XM4 | Bus4 | 건반 모니터 |
| XM5 | Bus5 | 드럼 모니터 |
| XM7 / XM8 | (메인) | Main L / R |

- [ ] Bus 전체 **Pre-fader** 설정 확인
- XM↔Bus는 순서대로 단정(XM1=Bus1 … XM5=Bus5). 실제 콘솔 라우팅이 다르더라도 운영상 영향은 거의 없음.

## 4. 메인 스피커 (JBL PRX ONE L/R, POST-fader)

> 밸런스: **전체 균등 + 보컬 살짝 위** (A-B 사이)
> 메인 스피커가 세션을 향함 → 보컬은 중앙에서 메인으로 자기 소리 모니터

- 모든 채널 → Main 적당히 (방 전체에 합주 깔리게)

!!! warning "잠정 — 합주 재검증 필요"
    **채널 페이더 기준값 (잠정)**: 악기 **-15** / 보컬 **-10** (보컬이 메인에서 5dB 더 도드라짐)

- 메인 밸런스 = **채널 페이더 위치**로 조절 (post-fader). 모니터 send와 독립.
- ※ 페이더는 메인으로 가는 양만 정함. 각 모니터의 보컬/악기 양은 3번 Sends 값이 따로 관리.

### 하울링(피드백) 대비

- [ ] 보컬 마이크는 메인 스피커 **정면 축 살짝 비낀 위치**
- [ ] 보컬 로우컷(100~120Hz) ON 확인
- [ ] 하울링 주파수 뜨면 메인 EQ 좁은 Q로 컷 (→ 6. 부밍처리)
- 마이크는 다이나믹(카디오이드) 권장

### 메인 스피커 본체 설정 (PRX ONE)

!!! warning "잠정 — 합주 재검증 필요"
    - CH1 고정, Main Volume: **-10dB** (잠정 — 혼자 확인, 합주로 재검증 필요)

- CH1 Gain: **0dB**

## 5. 보컬 리버브 (Bus13, FX) — ※확인 후 확정

!!! warning "잠정 — 합주 재검증 필요"
    현재 **B 구조(모니터에도 리버브)로 추정** → 내일 실물 확인 (미확정)

확인 항목:

!!! note "현장에서 채울 것"
    - [ ] ch18(보컬) → Bus13 send 되는지 (양: ____)
    - [ ] Bus13 → FX 엔진(리버브) 입력 연결됐는지
    - [ ] FX return 행선지: Main만? / 모니터(Bus1~5)에도?
      - **A** = 메인에만 (드라이 모니터, 깔끔/안전)
      - **B** = 모니터에도 (보컬 본인 리버브 들음, 복잡)
    - [ ] A로 단순화할지 B 유지할지 결정: ________

> 리버브 send는 **post-fader** (소리에 비례해 잔향 따라가야 함)

## 6. 부밍 / 하울링 처리 (나중 단계)

> 기본 세팅 끝난 뒤 진행

방법:

1. 메인 또는 모니터 버스 EQ에서 좁은 Q + **+6dB 부스트** 밴드 만들기
1. 주파수를 좌우로 쓸어가며 **웅~/삐~ 울리는 지점** 찾기
1. 찾으면 그 지점 **-3~6dB 컷**
1. X32 **RTA** 기능으로 스펙트럼 보면 더 빠름
1. 합주실 부밍은 보통 **100~250Hz** 대역

기록:

!!! note "현장에서 채울 것"
    - 부밍 지점 1: ____Hz, ___dB 컷
    - 부밍 지점 2: ____Hz, ___dB 컷
