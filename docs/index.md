# 스포렉스 합주실 PA 안내

스포렉스 합주실의 음향(PA) 사용을 돕는 안내 사이트입니다.

## 이 사이트를 만든 이유

이 합주실은 공용으로 쓰다 보니 세팅이 자주 흐트러졌습니다 — 모니터 스피커 전원이 꺼져 있거나, 스피커 볼륨이 바뀌어 있거나, 믹서 뒷편의 물리 연결이 달라져 있는 일이 반복됐습니다.

그래서 **2026년 5월 31일 기준으로 공용 기준 세팅(`02 mua-default-scenes` 씬)**을 만들어, 이용하시는 분들이 늘 같은 상태에서 시작할 수 있도록 정리했습니다.

이 조치들은 임의로 바꾼 것이 아닙니다. 합주실 **셋업 당시 건물 담당자에게 받은 초기 설정 문서(연결도)를 기준**으로, 실제 연결·세팅이 그 기준과 다르게 어긋나(망가져) 있던 부분을 **원래대로 되돌린** 것입니다.

**2026-05-31 기준 정상화 조치**

- 메인 스피커 **L 연결 정상화** (믹서 뒷편에 잘못 연결돼 있었음)
- **키보드 연결 정상화** (드럼 인풋에 연결돼 있었고, 중간 단인 스테이지박스에서 키보드1·2가 서로 바뀌어 있었음)
- **기타2 모니터 스피커 추가 셋업** (구석에 방치돼 있던 것)
- **모니터·메인 스피커 볼륨/세팅 통일**

사용 후에는 **`02 mua-default-scenes` 씬을 다시 불러와** 다음 분이 같은 상태에서 시작할 수 있게 해주세요.

## 어디로 갈까요?

- **처음 오셨거나 합주만 하실 분** → [사용 가이드](guide.md)
  악기 꽂는 자리, 씬 불러오는 법, 문제 생겼을 때만 담은 짧은 안내입니다.

- **세팅을 관리하는 분(관리자)** → [스펙 문서](spec.md)
  게인값·연결 케이스·매뉴얼 근거까지 담은 상세 문서입니다.
