# guide-authoring — 이용자용 가이드 작성

믹서를 모르는 이용자가 합주 직전에 급히 보는 안내를 쓴다. 메시지는 하나: **"씬 불러오기 + 본인 자리에 꽂기"만 하면 된다.**

## 1순위 규칙: 미검증 잠정값·빈칸 금지 / 확정 운영값은 허용

기준은 **"확정이냐 미검증 잠정이냐"**다 — "게인 수치냐 아니냐"가 아니다. 이용자는 검증 안 된 값을 따라 하면 세팅이 꼬이지만, **확정된 운영값**은 알아야 본인 악기를 제대로 연결·조정할 수 있다.

- **spec-editor의 "확정 정보 목록"에 있는 것만 쓴다.** 목록에 없으면 안 쓴다.
- **넣어도 되는 것 (확정 운영값)**: 케이스별 확정 게인값(앰프 인풋 0 / 멀티 XLR 직결 -10 / 멀티→앰프 Return 직결 +10 등), 앰프 노브 위치(베이스 마스터 12시, 기타 클린·크런치 12시 / OD 10시), 본체 노브 기준 위치. 이용자가 본인 악기를 맞추는 데 꼭 필요하다.
- **절대 금지 (미검증·잠정·빈칸)**: "잠정" 표기된 값(모니터 분배 -10/-15, 메인 페이더 악기 -15/보컬 -10, 메인 본체 -10dB), 빈칸 `______`, 미확정 구조(리버브 B 구조), 관리자만 만지는 컴프/게이트/EQ 파라미터.
- 확정 여부가 애매하면 **빼는 쪽**을 택한다(누락이 위험보다 안전).

이게 guide-writer의 핵심 판단이다 — 게인 수치를 무조건 빼는 게 아니라, **미검증 값만** 걸러낸다.

## 쉽게 쓰는 법

- 전문용어 금지: "Pre-fader Bus" → "본인 자리 앞 바닥 스피커". "48V" → (아예 언급 안 함, 건드리지 말라고만)
- 명령형 단문, 번호 단계, 표 1~2개. 한 화면에 들어오게.
- 스크린샷이 필요한 곳은 `![씬 불러오기 화면](img/scene-load.png)` 자리표시자를 남긴다.

## guide.md 골격

```
1. 시작 전 핵심   — 기본 씬 불러오기 + 끝나면 다시 불러오기 / 함부로 만지지 말 것
2. 씬 불러오기    — 어느 씬이 "이 가이드 기본 씬"인지 + 다른 씬(최초세팅/구버전) 구분 안내
3. 악기별 가이드   — 악기마다 앰프·본체 노브 기준 위치, (기타) 연결 방식별 게인 조정 방법
4. 스피커 조절    — 메인(메인 노브 + 채널 페이더 미세조정) / 모니터(버스 셀렉트 후 채널 페이더)
5. 문제 생기면    — 빠른 체크리스트(뮤트? 페이더? 씬 다시 로드) + 관리자 연락처
6. 하지 말 것     — !!! danger 박스 (상세)
```

> **이미 세팅된 정보는 싣지 않는다.** 입력 채널 매핑처럼 씬에 이미 잡혀 있어 이용자가 손댈 필요 없는 정보는 가이드에서 뺀다(혼란만 준다). 이용자가 **직접 맞춰야 하는 것**(본인 악기 노브 위치, 연결 방식별 게인, 스피커 음량 조절)에 집중한다.

## index.md 골격

- 한 줄 소개: "스포렉스 합주실 PA 안내"
- 갈림길 2개: ［처음 오셨나요 → 사용 가이드］ / ［세팅 관리 → 스펙 문서］ (상대 링크 `guide.md`, `spec.md`)
- 장비 한 줄: X32 Producer / DHR12M ×5 / PRX ONE ×2

## admonition 사용

- 금지 사항: `!!! danger "건드리지 마세요"`
- 도움 팁: `!!! tip`
- 주의: `!!! warning` (단, 잠정값을 넣으라는 뜻이 아님 — 운영상 주의만)

## 갱신 모드 (재호출)

- 기존 `guide.md`/`index.md`가 있으면 읽어서 갱신.
- spec에서 잠정→확정된 값이 와도, **이용자에게 실제로 도움이 될 때만** 추가한다. 대부분의 게인 수치는 확정돼도 guide엔 불필요("건드리지 마라" 유지).

## 외부 밴드 섹션 범위

가볍게 1섹션. "기타는 앰프 인풋에 꽂으면 됩니다 / 자리나 게인을 바꿔야 하면 직접 만지지 말고 관리자에게 연락하세요" 수준. 복잡한 연결 케이스 분기는 spec으로 안내(링크).
