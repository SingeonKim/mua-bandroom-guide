## guide 전달 가능 (확정·안전)

### 채널/악기 매핑 (확정)
- 표준 악기 배치 ch 매핑:
  - 베이스 = **ch8**
  - 키보드1 (Roland Fantom 07) = **ch9/10**
  - 키보드2 (Roland RD-2000) = **ch11/12**
  - 기타1 (Marshall JVM 410H) = **ch13**
  - 기타2 (Marshall JVM 410C) = **ch14**
  - 보컬1/2/3 (Shure PGA57 ×3) = **ch17/18/19**
- 드럼은 **마이킹 안 함(생음)** → 믹서 채널 없음
- 보컬 마이크 종류 = Shure PGA57 ×3 (다이나믹, 48V 안 씀) — 사실 정보로만 노출 가능

### 스피커 구성 (확정)
- 모니터 = **각 세션 앞 바닥 스피커** (Yamaha DHR12M ×5)
- 메인 = **세션을 향하는 스피커** (JBL PRX ONE L/R ×2)
- 모니터(Pre-fader)와 메인(Post-fader)은 독립 — "둘이 따로 논다"는 개념만 전달 가능 (수치 없이)

### 운영 원칙 (확정)
- **"건드렸으면 씬 다시 불러오기"** — 복구는 씬 불러오기 한 번
- 씬 불러오기로 항상 기본 상태 복구 가능
- 자리·게인 재배치가 필요하면 **관리자에게 요청** (사용자가 게인 직접 만지지 않음)

### 안전수칙 (확정 — guide에 꼭 넣을 가치 있음)
- **튜브앰프(Marshall JVM 410H/410C)는 캐비넷/스피커 연결 유지** — 스탠바이 끄고 PA만 쓸 때도 스피커선 빼지 말 것 (부하 없이 켜면 출력 트랜스포머 손상)
- **페달을 앰프 Power Amp Insert(Return)에 직접 꽂지 말 것** — 단순 페달 사용자는 앰프 인풋에 연결
- 안 쓰는 보컬 마이크는 뮤트/페이더 다운 (하울링 방지) — 사용자 행동 안내로 적합

### 외부 밴드/일반 이용자 연결 안내 (확정·동작)
- 기타: **앰프 인풋에 꽂기** (Case A) — 가장 단순·안전한 기본 연결
- 베이스: 앰프 인풋 + 캐비넷+PA 동시 (기본·권장)
- 베이스 캐비넷 끄려면 헤드폰 잭(1/8")에 더미 플러그 (필요시만)
- 모니터 스피커 본체: Input1 = LINE, D-CONTOUR = MONITOR (만질 일 거의 없음 — 노출은 선택)

## guide 전달 금지 (잠정·위험·빈칸)

### 모든 게인(dB) 수치 — 절대 금지
- 베이스 PA 채널 게인 **+21dB** / 페이더 -15 / 마스터 12시
- 기타 케이스별 게인: D 직결 **-10dB** / A 앰프인풋 **0dB** / B 4CM **-10~0dB** / C Return 직결 **+10dB**
- 페이더 -15 기준 등 모든 기준 페이더 수치

### 잠정값 — 절대 금지 (합주 재검증 전)
- 모니터 분배 수치: 본인 -10 / 타 -15, Bus 마스터 0~-10
- 메인 채널 페이더: 악기 -15 / 보컬 -10
- 메인 본체 PRX ONE Main Volume **-10dB**
- 리버브 구조 (B 구조 추정 — 미확정)

### 세부 파라미터 — 금지 (관리자 전용)
- 컴프 파라미터 (Threshold/Ratio/Attack/Release/Makeup/Knee)
- 게이트 파라미터 (Threshold -55dB, Range, Attack/Hold/Release 등)
- EQ 대역별 보정값 (150~250Hz +2~3dB 등)
- HPF/하이컷 주파수, 48V on/off 세부

### 빈칸/미확정 — 금지
- 모든 `______` 빈칸 (채널 게인 기록, 베이스 멀티직결 게인, 부밍 지점 등)
- Bus → 물리 아웃풋 매핑 (XM1~5, 미확정)
- 하단 판넬 AES50/ULTRANET (확인 필요)
- 보컬 동시 사용 인원 (미정)
